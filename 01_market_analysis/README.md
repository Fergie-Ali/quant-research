# 01 — Market Analysis

## What this does
Loads real stock price data and computes moving averages,
daily returns, and rolling volatility using Python.

## Key concepts
- Moving averages (MA_20, MA_50, MA_200) and the signal vs noise tradeoff
- Daily returns and volatility clustering
- Effect of window size on smoothing

## How to run
pip3 install yfinance pandas numpy matplotlib
python3 stock_analysis.py

## Libraries
Python · pandas · numpy · matplotlib · yfinance
