# 04 — Options Pricing

## What this does
Generates buy&sell signals based on moving averages crossover strategy, computes cumulative returns and displays portfolio growth.

## Key concepts
- Generates buy/sell signal when two MA with different windows crossing
- Computing sharpe ratio and max drawdown
- Comparing strategy to buy&hold

## How to run
pip3 install yfinance pandas numpy matplotlib
python3 ma_crossover_strategy.py

## Libraries
Python · pandas · numpy · matplotlib · yfinance
